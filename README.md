# SandboCV

An application for you to learn about XR topics using Computational Vision.

## Getting started

You will need to be on Linux and have installed opencv, numpy and matplotlib.

## Usage

`s` Start OpenCV

`q` Quit

`j` Up

`k` Down

`enter` Confirm action

## Content

- Object tracking
- Object Recognition
- Object Reconstruction
- Image Segmentation
